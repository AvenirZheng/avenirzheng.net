I have a true passion when it comes to designing and codeing a bright new project, I'am currently available for freelance work, if there are more information on request, please feel free to get in touch with me. Anyway, I would like to make friends with all your guys.

Favorites.典藏

Books.嗜书如命

    • 重构HTML
    • 作者：（美）哈罗德
    • 摘要：
      前端领域发展的又一次见证，很多时候，多花一点点心思，便能实现良构的、具有可访问的网页，当然也要学会利用自动化工具。
    • 未来是湿的
    • 作者：[美] 克莱·舍基
    • 摘要：
      社会性力量的崛起
    • Web 标准实战
    • 作者：（美）锡德霍姆
    • 摘要：
      一本不错的入门手册，只是出版的太晚了。作为实战用书，更推荐嗷嗷的《Web标准设计》。
    • 众包
    • 作者：杰夫·豪
    • 摘要：
      讲述Web 2.0浪潮下新生产方式，理论案例结合，给人不错的启发。全书虽系统，仍略显的文笔拖沓，处处能见“复制、粘贴”的痕迹。
    • 第一团队：携程与如家
    • 作者：朱瑛石/马蕾
    • 摘要：
      “先把最危险，最有争执的利益说清楚，才能坦诚相见“，若季琦真如所说，两次创立新公司，两次平静放弃权力，其胸怀真让人尊敬。
    • 瞬间之美
    • 作者：霍克曼
    • 摘要：
      一篇通读，文字很简单，但能处处感受到作者对细节的斟酌，案例、解决方案，而更为重要的是它不断启发读者对用户体验的思考。
    • 精通 JavaScript
    • 作者：John Resig
    • 摘要：
      作者以一章专门讲述JavaScript的面向对象编程，作者剖析类库的专研态度也很值得读者学习。
    • JavaScript语言精粹
    • 作者：Douglas Crockford
    • 摘要：
      书很薄，确将JavaScript语言的特性讲得很细致，细细品味中。
    • JavaScript DOM高级程序设计
    • 作者：Samb-ells，J
    • 摘要：
      全面深入地介绍JavaScript方方面面，最佳实践、原型域、作用链、DOM、库，案例同样很精彩。
    • PPk谈JavaScript
    • 作者：Koch，P.P
    • 摘要：
      正如后记所说的，这本书应该是学习的开始，而不是结束，作者功力无容置疑，作者分享了大量开发经验，对可访问性、可用性的关注。
    • JavaScript DOM编程艺术
    • 作者：Jeremy Keith
    • 摘要：
      一本很适合没有编程基础的设计师的Dom Scripting入门书，对Dom标准、优雅退化，可访问性做了详细描述，是很好的启蒙书。
    • Web标准设计
    • 作者：刘杰（嗷嗷）
    • 摘要：
      很全面的一本书，在很多地方上，嗷嗷都有深刻见解，语义化、优雅退化的概念也时有穿插。
    • 精通CSS与HTML设计模式
    • 作者：Michael Bowers
    • 摘要：
      在图书馆掏的书，内容很详尽，需要掌握一定设计模式基础，书读起来很生涩，有很多新的想法展现在书中。
    • Web设计创新思维
    • 作者：Adams等
    • 摘要：
      在图书馆借来看的书，多位大师的实战经验浓缩成一本书，感觉内容安排比较杂乱，但作为提高技巧应该是本不错的书。
    • 精通HTML
    • 作者：Paul Haine
    • 摘要：
      HTML总是最基础却又最容易让人忽略，关于语义化、微格式、HTML5等，都做了详尽的介绍，书很薄，但受益匪浅。
    • 写给大家看的设计书
    • 作者：Robin Williams
    • 摘要：
      正在阅读，Robin法则对齐，重复，对比，亲密性“在打破规则前，必须了解规则”、“要有空白，而且空白也要组织。”
    • CSS实战精粹
    • 作者：Jeff Croft / Lan Lioyd
    • 摘要：
      很好的一本CSS实战书，技巧方法都很实用，终于知道holly hack原来是这样来的，呵呵。
    • 少有人走的路：心智成熟的旅程
    • 作者：M.斯科特.派克
    • 摘要：
      我们都在逃避责任，是一本很好的自省书籍，从今天就开始行动吧。
    • jQuery基础教程
    • 作者：斯威德伯格/查弗
    • 摘要：
      比较难得的一本介绍JavaScript框架的书籍，居然能在图书馆淘到这样一本书，看过后发现JQ真得太奇妙了。
    • 深入浅出JavaScript
    • 作者：海尔曼
    • 摘要：

    • 精通CSS——高级Web标准解决方案
    • 作者：Andy Budd
    • 摘要：
      从入门到精通，是我做了最多笔记的一本书，中英文版一一收录。
    • 网页设计与配色实例分析
    • 作者：黎芳
    • 摘要：
      向朋友借来看的一本书，比较全面地介绍网页设计，案例和理论结合。
    • CSS网站布局实录
    • 作者：李超
    • 摘要：
      中文CSS书籍的典范，通俗易懂，就从这本书开始，慢慢走向web标准
    • 网站重构：应用Web标准进行设计
    • 作者：Zeldman,J.
    • 摘要：
      绝对是一本洗脑的权威书籍，没有太多方法和技巧，只有不断强调改变观念，遵循网页标准。
    • 生而为赢
    • 作者：俞敏洪
    • 摘要：
      简简单单的一本书，确饱含智慧，每篇文章都是背诵的佳文，王强老师的朗读实在是好。
    • 赢在用户
    • 作者： Steve Mulder / Ziu Yaar
    • 摘要：
      讲授创建人物角色方法的图书，内容涉及一些用户体验的理论，仍在反复攻读。
    • 交互设计之路
    • 作者：库帕（Coper,A.）
    • 摘要：
      主要阐述交互设计的重要性、理念的图书，内容比较概括。
    • HTML XHTML权威指南
    • 作者：Chuck Musciano
    • 摘要：
      遇到不了解的标签时，就会翻一下，感觉是前端必备的工具书之一
    • CSS禅意花园
    • 作者：Dave Shea/Molly
    • 摘要：
      感叹CSS设计师的无穷创意，一本(X)HTML CSS入门的好书，大量的案例，各种解决问题的思路，值得珍藏。
