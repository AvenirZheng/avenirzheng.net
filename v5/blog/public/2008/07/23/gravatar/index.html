{ Gravatar } 博客头像

　　折腾了一天，终于把博客的评论功能优化了（其实也就加了Ajax Comments-Reply借个插件，以后可以回复评论了）

　　离题了，貌似在国内Gravatar是非一般的不普及，我也是不久前才用上，呵呵。那为什么还要介绍Gravatar呢？因为wordpress2.5（现在的博客系统）内置了Gravatar功能，最关键还是装了Ajax Comments-Reply后，自动默认被Gravatar上了，so，科普一下，高手略过….

　　坚持不忽悠，不卖弄的原则。其实只需要到Gravatar网站上去注册一个账号，填邮箱，收激活信，登陆，上传头像，OK,done(其实还要等待审核，会为我们分G普通级、PG辅导级、R和X 为限制级，够专业吧)。

　　完成后，以后上博客评论，只要填上邮箱，就能出现你的头像，前提是该博客用上了Gravatar（wp2.5的推出，相信Gravatar应该能普及的）

　　所有说，Gravatar类似于OpenID，作为互联网上统一的个人标识。
