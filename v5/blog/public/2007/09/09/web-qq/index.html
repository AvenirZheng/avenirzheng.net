{Web QQ} Web QQ 试用

腾讯再次推出网页版QQ，应该很多人还不知道吧。据我所知应该是第二版，腾讯好像一直没有打算推广的意思，这个也应该只是个测试版。 个人感受：
1.起码能登录；之前用Web QQ，一直无法登录，所以也就没有继续使用。
2.在IE下无法显示名字，而在FIREFOX下，显示正常；不知道这是不是我个人问题。
IE下…
firefox下…..
3.IE下经常短线，FIREFOX还可以；用QQ的人多会用IE吧。
4.功能薄弱；只能基本聊天，可能腾讯没打算重点开发吧。
5.设计上延续腾讯抄袭的习惯；设计成个人桌面应该是为了好适应吧，奇怪的是底部用上了VISTA的任务栏,背景却是MAC的墙纸；
个人很希望能用上网页版的QQ，可是腾讯又一次让我失望。

题外话：WEB QQ的背景跟我丑机的桌面是一样的，应该有上万人在用吧，我宿舍就有另一个，腾讯是否应该考虑下自己设计咧，别丢脸了。
